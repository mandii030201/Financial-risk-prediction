# Financial Risk Prediction System

## Project Overview
This project aims to predict the probability of loan default using statistical and machine learning models.

## Objective
To identify key factors that increase credit risk and build models that assist financial institutions in decision-making.

## Tools & Technologies
- Python
- R
- Logistic Regression
- Random Forest

## Project Structure
data/ – raw datasets  
notebooks/ – analysis and modelling  
outputs/ – results and figures
